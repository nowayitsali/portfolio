# Portfolio

My personal portfolio site.

## Stack

- Next.js (App Router) + React + TypeScript
- Tailwind CSS
- shadcn/ui
- Motion
- next-themes

## Develop

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Build

```bash
npm run build
npm run start
```

## Content

Site content (profile, experience, projects, skills, education) is in `src/lib/data.ts`. The resume PDF and portrait image are in `public/`.
